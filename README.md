#Git main
#Git practice with branches

- git branch [branch]
- git checkout [branch]
- git checkout [branch] -b
- git merge [branch]
- git branch -d [branch]
- git push origin [branch]

### REMOTE REPOSITORIES

- git remote add [name][url]
- git push [remote][branch]
- git log --author=[name]
- git log --pretty=oneline
- git show[commit]
- git blame[file]
